# Fyu
Fyu is a lightweight command-line utility for managing tasks and notes without leaving the terminal workflow.

## Features
- Task CRUD & Tracking.
- Task Detail Viewers.
- Data Import/Export.
- Task Search & Notes Management.
- And more features.

## Installation
Quick install:
```bash
git clone https://github.com/Zeronetsec/Fyu
cd Fyu
chmod +x install.sh
./install.sh
```
For more detailed installation and uninstallation instructions, see [.docs/install_and_uninstall.md](.docs/install_and_uninstall.md).

## Usage Example
``` bash
fyu --create new_task
fyu --rename new_task test
fyu --export tasks.zip
fyu --import tasks.zip
fyu --list
```
And more commands.
